# MSH2 - Minecraft Server Hub

简洁美观的 Minecraft 服务器管理工具，支持多服务器集中管理。

![主界面](图片路径1)

## 功能特性

- 多服务器集中管理
- 一键启动/关闭服务器
- 终端控制台实时查看
- 配置管理
- Java 设置
- 游戏规则配置
- 日志管理
- 星标收藏服务器

## 使用说明

1. 添加服务器：点击"添加服务器"按钮，选择服务器核心文件
2. 管理服务器：在服务器列表中点击"管理"按钮进入详情页
3. 启动服务器：在详情页点击"启动"按钮
4. 查看日志：在终端控制台实时查看服务器输出
