# MSH2 - Minecraft Server Hub

简洁美观的 Minecraft 服务器管理工具，支持多服务器集中管理。

## 核心优势

### 优雅直观的界面设计
- 采用现代化的毛玻璃风格侧边栏与透明背景
- 流畅的页面切换动画，支持多种过渡效果（滑动、淡入淡出、缩放等）
- 自定义标题栏，无边框窗口设计
- 精致的状态指示器和交互动效

### 强大的多服务器管理能力
- 支持同时管理多个 Minecraft 服务器
- 服务器星标收藏功能，快速访问常用服务器
- 智能排序系统，自定义服务器展示顺序
- 服务器状态实时监控（运行中/已停止）

### 完善的进程管理
- 一键启动、关闭、强制关闭服务器
- 进程守护机制，支持断线重连
- 自动检测已运行的服务器进程并重新关联
- 优雅关闭服务器，自动发送 stop 命令

### 实时终端控制台
- 实时显示服务器输出日志
- 支持向服务器发送命令
- 日志文件自动保存
- 可配置日志显示行数和大小限制
- 错误日志高亮显示

### 灵活的配置管理
- 可视化编辑 server.properties 配置文件
- 自动识别服务器版本并匹配对应的游戏规则
- 支持在线获取最新游戏规则数据
- 智能版本匹配，自动选择最适合的游戏规则版本

### 智能内存管理
- 支持手动设置内存参数（最小/最大内存）
- 自动内存模式，智能分配服务器内存
- 支持自定义 Java 路径
- 可配置额外的 JVM 启动参数

### 自动更新系统
- 启动时自动检查更新
- 支持强制更新和可选更新
- 增量下载更新包
- SHA256 校验确保文件完整性
- 显示详细的更新日志

### 数据持久化
- 服务器配置自动保存到本地 JSON 文件
- 支持自定义背景图片
- 动画效果偏好设置记忆
- 窗口状态恢复

### 系统托盘集成
- 最小化到系统托盘
- 托盘图标快速操作
- 后台运行不占用任务栏
